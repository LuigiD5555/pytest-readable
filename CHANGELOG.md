## [0.1.0] - 2026-03-09

### Added

- Initial release of **pytest-readable**
- `@readable` decorator for attaching structured metadata to tests
- Human-readable CLI summaries
- Hierarchical test tree rendering
- Markdown documentation export
- CSV and JSON structured exports
- CLI helper command `readable`
- Multi-language output support (English and Spanish)

### Features

- Living documentation generated from test metadata
- Exportable specifications derived from test suites
- pytest-native plugin integration

### Notes

This initial release focuses on turning pytest test suites into readable specifications and exportable documentation.

Future releases may expand:

- language support
- export formats
- richer metadata structures
- tooling for documentation-driven and TDD workflows
